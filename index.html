Some stories are better told using motion graphics rather than still. This site is a utility for helping you create animated data visualizations using Tableau Public and telling stories in a unique engaging manner.

Getting Started

  1. Create a viz on Tableau public that can be animated using a paramter like so
  2. Visit the Create page and enter your info
  3. Enjoy your viz!
